# AudioSlice Pro

Professional Android audio recording and editing app with advanced voice isolation and noise reduction.

## Features
- Real-time voice isolation & noise suppression
- Professional audio editing (cut, trim, merge, split)
- Speed/pitch control
- Effects: Normalize, Voice Enhance, Reverb, Bass Boost
- Visual waveform editor with FFT spectrum

## Architecture
- Kotlin + MVVM + Hilt DI
- FFmpeg + WebRTC audio processing
- Material Design 3 UI
